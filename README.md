# Web / HTTP utilities in Go

Features:

-   [x] Static website router (http.Handler)
-   [x] Backend + frontend router (http.Handler)
-   [x] Status & response body size recorder (http.ResponseWriter)
-   [x] RespondJSON / RespondText
-   [x] HTTP router (based on gorilla/mux)
-   [ ] CORS middleware
-   [ ] Panic recovery middleware
-   [ ] Logging middleware
-   [ ] Rate limiting middleware
-   [ ] Authorization middleware
